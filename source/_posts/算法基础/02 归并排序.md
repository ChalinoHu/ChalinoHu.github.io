# 归并排序 

```c++
int tmp[N];
void merge_sort(int q[], int l, int r){
    if(l >= r) return;
    
    int mid = l+r >> 1;
    
    merge_sort(q, l, mid), merge_sort(q, mid+1, r);
    
    int k = 0, i = l, j = mid+1;
    while(i <= mid && j <= r){
        if (q[i] <= q[j]) tmp[k++] = q[i++];
        else tmp[k++] = q[j++];
    }
    while(i <= mid) tmp[k++] = q[i++];
    while(j <= r) tmp[k++] = q[j++];
    for(i = l,k=0; i <= r; ++i){
        q[i] = tmp[k++];
    }
}
```

归并算法是先划分，再归并，归并期间相关操作类似两个有序单链表的排序。递归函数总重返回的是q在l和r之间的有序部分，在递归函数每次出栈进行归并操作。

# 模板

```c++
void merge_sort(int q[], int l, int r)
{
    if (l >= r) return;

    int mid = l + r >> 1;
    merge_sort(q, l, mid);
    merge_sort(q, mid + 1, r);

    int k = 0, i = l, j = mid + 1;
    while (i <= mid && j <= r)
        if (q[i] <= q[j]) tmp[k ++ ] = q[i ++ ];
        else tmp[k ++ ] = q[j ++ ];

    while (i <= mid) tmp[k ++ ] = q[i ++ ];
    while (j <= r) tmp[k ++ ] = q[j ++ ];

    for (i = l, j = 0; i <= r; i ++, j ++ ) q[i] = tmp[j];
}
```
